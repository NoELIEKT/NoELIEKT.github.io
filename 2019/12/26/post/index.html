Chrismas CTF 2019 baby seccomp

open은 이미 프로그램에서 열어줬으므로, 문제는 read와 write이다.

  • write

    error based shellcoding으로 1바이트씩 비교하여 해결할 수 있다.

  • read

    • A > 0x40000000 이라서 0x4000000을 막지 않는다. eax에 0x4000000를 넣으면 인티저 오버플로우로 read를 syscall로 부를 수 있다
    • mmap 매개변수 fd에 3을 주면 파일 내용을 읽어올 수 있다.

EXPLOIT

IO가 조금 꼬여서 고생하긴 했는데 read로 bss에 읽어서 푼 익스코드다. 익스 작성 시간보다 IO 때문에 고생좀 했다.
